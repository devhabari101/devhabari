Linapokuja swala la huduma ya mazingira ya uendelazaji wa code kwenye mtandao wa internet au niseme mawinguni
huduma mbili zina nijia kwa haraka github codespace na gitpod 

Github ni mtandao wa kijamii mashuhuri sana kwa waendelezaji code hasa kwa sababu jukwaa hilo
lili tengenenzwa kwa ajili ya waendelezaji na hivyo kua na huduma kadhaa kwa ajili ya waendelezaji na 
moja ya huduma ambayo imekua marufu zaidi hata kwa watu wasiyokuwa waendelezaji wa code ni huduma ya kuhifadhi 
code kwa matoleo na matawi kwenye mti wa hadhina **"reposatory"**, pamoja na michakato mingine ambayo inaweza kufanyika bada ya kuhifadhi hizo code kwenye hadhina 
kama vile kuchukua hatua flani **git action**.

kwenye makala haya tutakwenda kuangalia huduma ya mazingira ya kuandikia na kuendleza code inayo tolewa na github maarufu kama codespace. Tutaagalia jinsi ilivyo rahisi kutengeneza CodeSpace na jinsi ya kuifanyia setting ili iendane na mazingira yako. Chanzo cha code kinapatikana hapa

CodeSpace Nimazingira ya uendlazaji ambayo yanafanya kazi katika cointainer ambayo mwenyeji wake ni virtual machine ambayo itakuwa imeunganishwa na hadhina yako.

Tumekua tukitumia macointerna kuendeleza na kusafirisha code ili kutatua changamoto ya code zilikuwa zinafanya kazi kwangu, usanifu wa huduma ndogo ndongo **microservice architecture** hauepukiki hasa kwa sababu ya ubora wa ufanyaji kazi wa application ambazo zime jegwa kwa usanifu huu, kwenye team kuwa na mazingira yauendelzaji yanayo fuanana setting zake inachagiza 
kasi ya ufanyaji kazi lakini uwezo wakufanya kazi popote na kwenye kifa choochote 
kwenye machine ambayo inauwezo mkubwa labda kuliko kifachako na bila malipo ya internet kwenye hiyo machine hiyo inakuwa zaidi ya sababu ya kutumia mazingira haya wakati wa uendelezaji code. 

Codespace imejengwa mahususi kuweza kuenda
sambamba na mahitaji ya project yako pamoja na waendelezaji Kwa kutumia mafaili ambayo yamefanyiwa setting maalumu **configuration** pamoja na chanzo cha code ambazo zitatumika kutengeneza codespace kwa matoleo na ambayo unaweza kuitumia mara kwa mara na setting zake zina fuanana kwa watumiaji wote.

Mtu yoyote yule anaweza kutumia codespace chamsingi uwe tu na accout github 

# Jinsi ya kutengeneze codespace

ilikutumia codespace itabidi uingie github na account yako kama huna sajili, 
zingatia na anza na kutengeneza hadhina "repo" ambayo nataka kuiunganisha na codespace kama tayari unayo hadhina
ambayo unataka kuianganisha na codespace huna haja ya kutengeneza hadhina mpya lakini kama huna endelea kamaifuatavyo

![github_repo](https://i.ibb.co/qBr5pV8/github-codespace-rep0edit2.jpg)

![github_repo](https://i.ibb.co/GWCcBYp/github-codespace-rep1.jpg)

kwavile tayari ninayo hadhina tuanze mojakwamoja na kutumia codespace, fungua profile yako nauende kwenye kitufe cha codespace kama hivi

kuna template ambazo tungeweza kuanza nazo kutegemea na mazingira yako mimi sitachagua templete yoyote ile kwani templete ambayo nitaanza nayo
niile ambayo ni chagua lamsingi endapo huta chagua templete yote ile, yayewe inakuja ikiwa imewezesha na node, python, docker hii hasa inatufaa
kwenye mazingira ya maktaba hii ambapo tunahitaji kutumia docker zaidi. kumbuka unaweza ukatengeneza templete yako na ukashare na
wengine kwenye soko la github
